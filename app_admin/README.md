# Fulltech Admin

App administrativa local creada con Flutter/Dart para gestionar la web de Fulltech.

## Ejecutar

```bash
flutter pub get
flutter run
```

## Credenciales demo

- Correo: `admin@fulltech.local`
- Contrasena: `admin123`

## Que incluye

- Login local de demostracion.
- Dashboard con resumen.
- Gestion local de ofertas.
- Gestion local de categorias.
- Seguimiento basico de solicitudes.
- Pantalla de ajustes preparada para conectar WhatsApp, redes y usuarios.

## Nota importante

Por ahora los cambios viven en memoria dentro de la app. Para administrar la web real se debe agregar un backend o una API que guarde cambios en base de datos o sincronice con los archivos del proyecto.
